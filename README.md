# Directorio de Personas
Aplicación de escritorio para registrar personas con su nombre y número telefónico.
Los datos se guardan en un archivo CSV en la carpeta `data/`.
## Características
Interfaz gráfica simple con tkinter
Agregar personas al directorio
Validación de campos vacíos
Datos guardados en CSV
Estructura profesional (src/ y data/)
